Segurança

Construído para rodar trancado

Credenciais cifradas em repouso, autenticação forte e trilha de auditoria completa — auditado contra o OWASP Top 10.

Credenciais cifradas (AES-256-GCM)

Senhas de RouterOS e tokens de controlador nunca ficam em texto plano no banco.

2FA TOTP por usuário

Compatível com Google Authenticator e Authy. JWTs invalidados no logout e na troca de senha.

Webhooks assinados (HMAC-SHA256)

Endpoints de UP/DOWN para integradores externos como Zabbix e Nagios, com verificação timing-safe.

Rate limiting & audit log

Login limitado por IP, controle de acesso por papel (Admin/Operador/Viewer) e registro de toda alteração.

lib/crypto.ts
// Credenciais cifradas antes de persistir
export function encrypt(plaintext: string) {
  const iv  = randomBytes(12);
  const key = validateKey(env.ENCRYPTION_KEY);
  const c   = createCipheriv(
    "aes-256-gcm", key, iv,
  );
  const enc = Buffer.concat([
    c.update(plaintext, "utf8"),
    c.final(),
  ]);
  return { iv, enc, tag: c.getAuthTag() };
}
